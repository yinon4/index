### Heading ID ‚Äî ‚ùå unsupported

Curly braces in MDX escape to JavaScript, so they can't be used to assign ids.

```md

```

Use JSX instead

```jsx
<h1 id="custom-id" style={{ color: "green" }}>
  My Great Heading
</h1>
```
